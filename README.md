GOTO Conference Zurich 2013 Java on Azure Webapplication Sample
===============================================================

Deployed Cloud version on Windows Azure should be available at [BookStore](http://bookstoregotozurich.cloudapp.net/bookstore-web)

This project contains the source code for the demo prepared for the [GOTO Zurich Conference 2013](http://gotocon.com/zurich-2013/) 
for the session [Developing Professional Solutions with Java, Spring & NoSQL on Windows Azure](http://gotocon.com/zurich-2013/presentation/Developing%20Professional%20Solutions%20with%20Java,%20Spring%20&%20NoSQL%20on%20Windows%20Azure) by 
Jürgen Mayrbäurl and Mario Szpuszta
